# GitHub Copilot workshop: presenter-led příběh od plánu ke governed delivery

Tento workshop je optimalizovaný pro presenter-led technický průchod s live demo ukázkami. Hlavní message je, že GitHub Copilot už není jen o autocomplete nebo chatu, ale o **governed multi-agent engineering workflow** napříč VS Code, Copilot CLI, GitHubem, GitHub Actions a provozem.

- Rychlé ukázky a kontext nad repozitářem: volba modelu, porozumění codebase a rychlé ukotvení v tom, co v tomto repu skutečně je
- Customization stack: `AGENTS.md`, prompt files, custom agenti, hooks, skills a MCP, včetně vysvětlení kdy co použít
- VS Code custom agenti v praxi: planner, specialisté, handoffy a samotný koncept subagentů
- Copilot CLI jako plnohodnotná plocha: plan mode, autopilot, `/yolo`, `/fleet`, sessions, `/tasks`, `/resume` a background worktree izolace
- Governed delivery po coding agentech: PR flow, Copilot code review, security findings, diskuse nad autofix a workflow agenti v GitHub Actions
- Agentic workflows a operations: `gh-aw` jako aditivní automatizace a zakončení přes Azure SRE Agent a post-deployment pohled
- Volitelné odbočky pro publikum, které chce víc: generování dokumentace, vision, KQL/SQL, web research, MCP deep dives, Spark a další legacy demo scénáře
