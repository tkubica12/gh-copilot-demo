# Workshop GitHub Copilot: Od promptu k modernímu AI engineering workflow s agenty

Tento workshop ukazuje, jak GitHub Copilot zapadá do nové generace AI-native engineering workflow napříč VS Code, CLI a GitHubem. V posledním roce se celý obor výrazně posunul a nástroje i modely překročily důležitý práh: agenti už nejsou jen užiteční asistenti, ale praktičtí spolupracovníci pro reálnou engineering práci napříč kódováním, testováním, review a automatizací.

- Základy a rychlé ukázky: volba modelu, chat, kontext nad codebase, research a úpravy více souborů
- Agentic delivery ve VS Code a Copilot CLI: agent mode, `/fleet`, subagenti, custom agenti a iterativní smyčky testuj-a-oprav
- Spec-driven development v praxi: AGENTS.md, prompty, PRD a specifikace a přístup ve stylu spec-kit pro spolehlivou end-to-end práci
- Skills a MCP dohromady: skills pro silné lokální vedení a opakovatelné schopnosti, MCP pro napojení agentů na vzdálené nástroje a živé systémy
- Agent HQ a cloud orchestrace: mission control, background a cloud agenti, third-party agenti jako Claude a Codex, sdílená paměť a volba izolace workspace
- Nejen vývoj: Copilot code review, security a autofix, agentic workflows v GitHub Actions a Azure SRE Agent
